# The sum of the first n squares

One of my favorite math problems is this: Find a formula for the sum of the first n squares.

In other words, find a formula for $1^2 + 2^2 + 3^2 + \cdots + n^2$.

I'm not going to give the answer, or the solution, because I know that a lot of people want to figure out the solution themselves.

But I can tell you about my journey in solving this problem.

I tried to solve the problem many years ago... and I couldn't!

I ended up looking up the solution on the internet.

I read the solution, and I even typed up my own solution to the problem.

A year later, I revisited the problem, and I wanted to test whether I remembered the solution.

Unfortunately, I did not remember the solution!

I had taken the time to type up a pretty solution to the problem, a year earlier. But on that day, I didn't remember the solution.

So I was faced with the philosophical question: How can I remember a solution to a famous problem like this one?

I realized that there is often a key to solving a problem. Many math problems have a key.

The key to a problem unlocks the problem.

So if I could remember the key to solving the problem, then maybe I could remember the entire solution.

I wanted to share this anecdote, because not only did I learn how to solve this problem... I also learned how to remember a solution.

Sometimes, you just have to remember the key to solving a problem, and then it's easy to come up with the solution.

You can form a really nice description of the key to solving a problem, and use that description to solve the problem.
